Mesh specification and upload

Structs

Mesh

A reference to a GPU mesh object that can be drawn by a Painter supporting the associated resource, vertex, material, and primitive combination. The memory cost of this object is negligible, since (unlike MeshSource) it is a cloneable reference to GPU allocated resources.
MeshSource

A mesh storage object and builder that sent as drawable geometry to the GPU. Once uploaded, this mesh can only be rendered using a Painter whose associated style supports the given vertex and material type (<V, M>). In addition, the painter must be setup for this mesh's primitive type.
